# Changelog

Tous les changements notables de ce projet seront documentés dans ce fichier.

Le format est basé sur [Keep a Changelog](https://keepachangelog.com/fr/1.0.0/),
et ce projet adhère au [Semantic Versioning](https://semver.org/lang/fr/).

## [0.1.0] - 2025-11-23

### Ajouté
- Application web pour réaliser un mini audit des startups
- Questionnaire interactif avec 25 questions réparties en 5 thématiques
- Graphique radar pour visualiser les résultats
- Système de scoring par catégorie et score global
- Messages de feedback personnalisés selon les scores
- Formulaire de contact
- Design responsive pour mobile, tablette et desktop
- Support des questions en format YAML

### Thématiques
- Business Model (5 questions)
- Produit (5 questions)
- Go-to-Market (5 questions)
- Organisation (5 questions)
- Financement (5 questions)
